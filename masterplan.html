5 Wix 적용 단계별 가이드

Wix 에디터에서 직접 적용하는 방법 — 코딩 불필요

1

랜딩 이미지 카드에 아이콘 오버레이 추가

Wix Editor에서 3개 이미지(나파밸리, 페블비치, 금문교)를 클릭 → Add Element → Icon 선택 → 이미지 위에 배치 → 투명도 조절로 아이콘+이미지 병행 표시

2

Wix 블로그 카테고리 설정

Wix Dashboard → Blog → Categories → "Wine", "Golf", "San Francisco" 3개 카테고리 생성 → 기존 포스트에 태그 적용 (Pebble Beach → Golf, Wineries → Wine)

3

카테고리 페이지 링크 연결

랜딩의 3개 이미지 카드 각각 → 우클릭 → Link → Web address → 제작한 블로그 HTML URL 또는 Wix 카테고리 URL로 연결

4

독립 HTML 페이지를 Netlify로 배포 (선택)

netlify.com → "Add new site" → "Deploy manually" → 제작한 HTML 폴더 드래그앤드롭 → 무료 URL 생성 → Wix 랜딩에서 해당 URL로 링크

5

Google Search Console 등록 (Phase 3)

Wix Dashboard → Marketing & SEO → Google Search Console → 사이트 소유권 확인 → 사이트맵 제출 → 블로그 포스트 검색 노출 시작
